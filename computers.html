I've been really into computers recently. like a LOT... I'm working on a lot of computers at the moment, and the most recent one I got is a Lenovo Ideapad running Windows 7, that I'm going to replace the battery for and get old photos out of.

I use arch btw (insert nerd emoji here)

I've been using arch on my laptop and desktop for about like 3 months... i think... maybe more, ANYWAYS, I love using it because I get bored really easily, and call me weird but, I like the amount of troubleshooting that comes with Arch. It gives me something to do that I feel productive doing, and it helps me learn more about how an operating system like Arch works. The only plus side I have to give to windows is the convenience for when I don't feel like problem solving, and the much much better support for random problems, and much more software support. When I was modding my xbox 360, i had to use my old laptop that had windows 11 and it made it so much easier.
